About

Saya merupakan orang yang suka untuk memecahkan suatu permasalahan yang membutuhkan penalaran. Ketika pertama kali saya mencoba masuk ke dalam dunia pemrograman, saya merasa seperti menemukan sesuatu yang benar-benar cocok dengan cara berpikir saya. Tantangan yang ditemui ketika membuat sesuatu membuat saya ingin menggali lebih dalam. Sejak saat itu, saya mulai mengembangkan diri dalam bidang pengembangan perangkat lunak.


Memasuki dunia perkuliahan menjadi loncatan penting bagi saya untuk mengasah kemampuan lebih dalam. Saya mendalami Java dan SQL untuk mengembangkan berbagai aplikasi desktop. Kemudian, saya mencoba untuk mempelajari website dengan menggunakan HTML dan CSS.


Saya memiliki ambisi untuk terus berkembang dan mengeksplorasi berbagai area software development ke depannya, seperti aplikasi Android, Internet of Things (IoT), dan Artificial Intelligence (AI). Tujuan saya adalah berkontribusi dalam menciptakan teknologi berkualitas yang dapat memecahkan masalah nyata dan memberikan dampak positif.
